Para poder jugar es necesario descargar ambos archivos, el Aixec.exe y el Aixec.pck, es necesario tener ambos archivos en una misma carpeta

El Ejecutable (Aixec.exe): El motor principal del juego.

El Paquete de Datos (Aixec.pck): Contiene todos los recursos.

La Aplicacion esta diseñada para poder visualizar el listado de cartas existentes, descargar el juego y visualizar el ranking de jugadores en modo LOGOUT.

En caso de registrarse y iniciar sesion este podrá visualizar en el menú las opciones para editar sus mazos usando las cartas del inventario, la opcion de obtener nuevas cartas mediante la compra de sobres, la opcion del chat donde podrás interactuar con otros jugadores y la vista de perfil para poder visualizar tus estadisticas básicas.
